# RMAnno Extra Annotation Dataset

This dataset provides additional annotation resources specifically for the [RMAnno](https://github.com/JiongmingMa/RMAnno) project, and can be used for annotation tasks in RNA modification research. Users are free to download and integrate it into their own analysis pipelines.

## 📖 Introduction

RNA modifications play an important regulatory role in epitranscriptomics. To help researchers functionally annotate and perform regional analysis of RNA modification sites more conveniently, we have built an **extra annotation dataset** based on the RMAnno framework. It covers various RNA modifications. This dataset can be used with the RMAnno main program, and can also be used independently.
